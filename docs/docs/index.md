# ContraDito

## Sobre

O **ContraDito** é um projeto em desenvolvimento da matéria **Métodos de Desenvolvimento de Software**, a qual tem como objetivo Tornar visível com dados reais a coerência entre o discurso e o voto de políticos do Brasil.
